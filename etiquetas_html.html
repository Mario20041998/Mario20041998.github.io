Enlace desde texto M.S.B.L

Para poner un enlace en un texto se utiliza la etiqueta "a". Dentro de esta ponemos el texto que queremos que sea un enlace.
Por ejemplo: Este texto te lleva a la página Instagram.

Enlace desde una imagen M.S.B.L

Para poner un enlace en una imagen debemos utilizar la etiqueta "a" y dentro de esta insertar la imagen en la que queramos poner el enlace con la etiqueta "img"
Por ejemplo, la siguiente imagen te llevará a la página Youtube.

Enlace absoluto desde un texto o imagen M.S.B.L

Un enlace absoluto es aquel que nos dirige a una página web ajena a nuestro sitio web. Se insertan igual que otro enlace.
Por ejemplo, los enlaces anteriores son absolutos, puesto que te dirigen a una página web diferente al de nuestro sitio web.

Enlace relativo desde un texto o imagen M.S.B.L

Un enlace relativo es aquel que te dirige a una página web dentro del sitio web en el que se encuentra el enlace. Se insertan igual que otro enlance.
Por ejemplo, los enlaces ubicados en la página principal y al final de cada página de mi sitio web son relativos.

Enlace que se abre en una ventana nueva M.S.B.L

Si quieres que un enlace se abra en una ventana nueva tienes que utilizar la etiqueta "a" con el atributo target="_blank".
Por ejemplo, si pinchas aqui se te abrirá twitter en una ventana nueva.

Un enlace interno a un sitio dentro de la misma página M.S.B.L

Para insertar un enlace que te direccione a un sitio dentro de la misma página web debes utilizar la etiqueta a con el atributo href="#id". El enlace te direccionará al sitio que hayas definido con la id.
Por ejemplo, si pinchas aquí, el enlace te direccionará al principio de la página.

Insertar una tabla M.S.B.L

Para insertar una tabla en tu página web, debes utilizar la etiqueta table. Después, tenemos que utilizar la etiqueta thead (para la cabeza de la tabla) y tbody (para el cuerpo de la tabla). Para añadir lineas, tendremos que utilizar la etiqueta th (para añadir columnas en el encabezado de la tabla) y td (para añadir filas en el cuerpo).
Por ejemplo, la siguiente tabla:

Nombre Ranking Puntos
Paquito 3 9375
Lebron 1 11785
Belasteguin 5 7365
