Propiedades del texto en CSS

Font Family genérica - M.S.B.L

Una familia de fuente genérica es una familia de fuente en general, por ejemplo "sans-serif" o "serif". Por ejemplo, este texto esta escrito con la fuente genérica "sans-serif".

Familia de fuente específica - M.S.B.L

Una familia de fuente específica es un tipo de letra concreta que pertenece a una familia determinada como por ejemplo "helvetica". Estos se ponen entre comillas dentro de la etiqueta style.
Este párrafo está escrito con la letra "courier", un tipo de letra de la familia de fuente "serif".

Tamaño de fuente (px) - M.S.B.L

Una de las formas para cambiar el tamaño de fuente es con el atributo "font-size" dentro de la etiqueta style. A este le daremos un valor en píxeles (px). Por ejemplo, este texto está escrito con una letra de 20 px.

Tamaño de fuente (em) - M.S.B.L

Tambień podemos cambiar el tamaño de la letra utilizando el atributo "em" en vez de px. "Em" sirve para poder modificar el tamaño con unas proporciones.

Por ejemplo, esta frase está escrito con una fuente un 200% más grande que el predeterminado por la página.

Estilo de fuente - M.S.B.L

El estilo de fuente se puede cambiar utilizando el atributo "font-style" dentro de una etiqueta style. En ella podemos poner la letra en cursiva, utilizando "italic".

Como ejemplo, esta frase está escrita en cursiva.

Altura entre líneas - M.S.B.L

La altura entre líneas se puede modificar utilizando el atributo "line-height" dentro de la etiqueta style.
Por ejemplo, la altura entre líneas de este párrafo es de 40px.

Alinear texto - M.S.B.L

Para alinear un texto a la izquierda, al centro o a la derecha tendrás que utilizar el atributo "text-align" dentro de la etiqueta style. Para alinearlo a la derecha, pondras text-align: right, y si lo quieres alinear al centro text-align: center.
Por ejemplo, el título de este documento está alineado en el centro.

Decoración del texto - M.S.B.L

Para decorar el texto tienes que usar el atributo "text-decoration" dentro de la etiqueta style.
