Lo primero que hay que hacer para poder usar CSS es escribir la etiqueta "style" debajo de la de "title". Dentro de ésta escribiremos el código de CSS.


Selección por etiqueta M.S.B

Escribiendo una de las etiquetas html podemos cambiar sus atributos de color, fondo... Por ejemplo, ahora mismo está aplicado el color rgb(207, 107, 107) a la etiqueta "h2" (encabezados).


Selección por id (#) M.S.B

Para cambiar los atributos de una sola etiqueta de html,lo hacemos mediante un id. La estructura es id="". Por ejemplo, el texto de párrafo de este apartado y mi nombre están cambiados con un id.


Selección por clase (class) M.S.B

Para cambiar los tributos de una única parte del código, debemos crear una clase que vaya dentro de la etiqueta html que queremos cambiar. La estructura es class="".

Por ejemplo, el color de esta frase esta modificado mediante una clase
